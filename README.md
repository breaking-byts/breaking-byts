### Research

Working with Dr. Sujay Deb on detecting malware through CPU performance counters. We train LSTM models on hardware telemetry to flag anomalies in real time, achieving ~0.90 balanced accuracy across 400K samples from the RaDaR dataset. Current focus: reducing feature extraction overhead for bare-metal deployment.
